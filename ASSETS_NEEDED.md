# Required Assets

Please add the following image files to the `public` folder:

## Images Required:

1. **photo.jpg** - Your professional headshot photo (circular crop recommended, will be displayed as 200x200px circle)

2. **linkedin-logo.png** - LinkedIn logo (48x48px recommended)

3. **github-logo.png** - GitHub logo (48x48px recommended)

4. **gmail-logo.png** - Gmail logo (48x48px recommended)

5. **bmp-logo.png** - Blackmore Partners logo (32x32px recommended)

6. **data-structures-icon.png** - Icon for Data Structures and Algorithms course (60x60px recommended)

7. **database-icon.png** - Icon for Database Systems course (60x60px recommended)

## Notes:

- All images should be placed in the `public` folder
- The photo will be automatically cropped to a circle
- Logos should have transparent backgrounds if possible
- Course icons can be simple representations (e.g., code brackets for DSA, database symbol for DB Systems)
