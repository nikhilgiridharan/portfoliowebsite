# Nikhil Giridharan - Portfolio Website

Portfolio website showcasing my work as a Data Engineer.

## Tech Stack

- React
- Vite
- CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is deployed on Vercel at [nikhilgiridharan.com](https://nikhilgiridharan.com)

## GitHub Repository

To set up this repository on GitHub:

1. Run the setup script:
   ```powershell
   .\setup-github.ps1
   ```

2. Create a new repository on GitHub at https://github.com/new
   - Repository name: `portfolio-website`
   - DO NOT initialize with README, .gitignore, or license

3. Connect and push:
   ```bash
   git remote add origin https://github.com/nikhilgiridharan/portfolio-website.git
   git branch -M main
   git push -u origin main
   ```
